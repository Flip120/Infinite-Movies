Infinite Movies
===================

#### How to use

#### **Install node.js dependencies**
```
npm install
```

#### **Install bower dependencies**
```
bower install
```
----------

### **Running**
You can run the server with :
```
node bin/www
```

### **Develop**
You can also serve the app with gulp:
```
gulp serve
```

or build for production:
```
gulp build
```
